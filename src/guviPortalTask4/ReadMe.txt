Java Advance - Exception & collection

Q1. Ramesh is developing a student management system for a university. In this system, you have a Student class to request student information. You are asked to help Ramesh to handle exception which can be occurred into program according to following scenarios:
a. Class Student with attributes roll no, name, age and course. Initialize values through parameterized constructor.
b. If the age of the student is not between 15 and 21 then generate a user-defined exception "AgeNotWithinRangeException".
c. If a name contains numbers or special symbols, raise exception "NameNotValidException". Define the two exception classes.

Q2. Create a class Voter(voterId, name, age) with parameterized constructor. The parameterized constructor should throw a checked exceptions "invalid age for voter"

Q3. Store name of weekdays in an array ( Starting form "sunday" at 0 index ) .Ask day position from user and print day name. Handle array index out of bound exception and give proper message if user enters day index outside range ( 0-6 ).

Q4. Create a HashMap where eys are student names ( strings ) and vales are their corresponding grades (integers). Create methods to add a new student, remove a student and display up a student's grade by name.

Q5. Use collection Classes to store integers. Create some methods for following functionslities.
a. Include functions for pushing elements onto the stack.
b. popping elements form the stack.
c. Checking if the stack is empty.
